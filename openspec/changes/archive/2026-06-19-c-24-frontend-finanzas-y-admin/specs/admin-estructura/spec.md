## ADDED Requirements

### Requirement: ABM Carreras

El sistema SHALL permitir crear, listar, editar y eliminar carreras (código, nombre, estado). Permiso: `estructura:gestionar`.

### Requirement: ABM Cohortes

El sistema SHALL permitir crear, listar (por carrera), editar y eliminar cohortes (carrera, nombre, año, vigencia). Permiso: `estructura:gestionar`.

### Requirement: ABM Materias

El sistema SHALL permitir crear, listar, editar y eliminar materias (código, nombre, estado). Permiso: `estructura:gestionar`.
