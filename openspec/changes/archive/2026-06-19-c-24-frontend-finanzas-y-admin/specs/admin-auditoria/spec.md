## ADDED Requirements

### Requirement: Log de auditoría con filtros

El sistema SHALL mostrar el log de auditoría con filtros por acción, materia, actor y rango de fechas. Permiso: `auditoria:ver`.

### Requirement: Dashboard de métricas

El sistema SHALL mostrar métricas de auditoría: acciones por día, comunicaciones por docente, interacciones por docente×materia, últimas acciones.
